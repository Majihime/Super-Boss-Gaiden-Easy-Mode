# Super Boss Gaiden Easy Mode
---

## About the Game
---

This is a freeware homebrew game written in 65816 assembly which can run in any emulator capable of running .sfc files. This is simply a modified version of the game which serves as an easy mode. Your character can now jump infinitely in air and endure infinite attacks from enemies. Be warned however, hazards such sa spikes can still deal damage.

The simplest way to play the game is to visit the releases page and run the pre-patched file.
